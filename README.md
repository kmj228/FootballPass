## 프로젝트 소개

K-League 팀 정보와 선수 데이터를 한곳에서 확인하고,<br/>
팀 커뮤니티 게시판과 경기 일정을 관리할 수 있는 Android 애플리케이션이다.<br/>

팀 선택부터 경기 라인업 구성, 게시판 활동, 버스 예약까지<br/>
팬과 팀 운영에 필요한 기능을 하나의 앱으로 제공한다.<br/>
Firebase 기반의 실시간 데이터 동기화로 여러 사용자가 동시에 사용할 수 있다.<br/>

---

## 주요 기능

- **팀 정보** — K-League 25개 팀 목록, 팀 상세정보 및 순위 조회
- **선수 정보** — 선수 상세 페이지, 선수 이미지 갤러리
- **경기 관리** — 경기 일정 조회, 라인업 구성, 필드 선수 배치 시각화
- **커뮤니티** — 게시판 생성·관리, 게시물 작성·수정·삭제, 댓글 시스템
- **버스 예약** — 원정 응원 버스 예약 및 좌석 관리
- **사용자 관리** — Firebase 인증 기반 회원가입·로그인, 프로필 사진 편집
- **이미지 처리** — 사진 촬영, 이미지 크롭, 클라우드 업로드

## 시작하기

### 사전 요구사항

- Android Studio Hedgehog 이상
- Android SDK 28 이상
- Firebase 프로젝트 및 `google-services.json`

### 실행

```bash
git clone https://github.com/kmj228/FootballPass.git
```

1. `google-services.json` 파일을 `app/` 폴더에 넣는다.
2. Android Studio에서 프로젝트를 열고 **Sync Now** 를 실행한다.
3. 에뮬레이터 또는 실기기에서 실행한다.
